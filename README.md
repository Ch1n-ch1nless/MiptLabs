## –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ –∞—Å–∏–º–ø—Ç–æ—Ç–∏–∫–∏ O(NlogN)

``` C

//[in]:     buffer_size         - —Ä–∞–∑–º–µ—Ä –º–∞—Å—Å–∏–≤–∞ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ 
//[out]:    OK / ERROR  = –≤—Å—Ç–∞–≤–∫–∞ –ø—Ä–æ–∏–∑–æ—à–ª–∞ —É—Å–ø–µ—à–Ω–æ / –Ω–µ—É—Å–ø–µ—à–Ω–æ.

    assert((heap->data != NULL) && "Pointer to \'heap->data\' is NULL!!!\n");
```

$$\displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil \frac{N}{2^{h+1}} \bigg\rceil O(h) = O(N\displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil \frac{h}{2^{h+1}} \bigg\rceil)$$

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º $\displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil \frac{h}{2^{h+1}} \bigg\rceil = \displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil (\frac{1}{2} * \frac{h-1}{2^{h}} + \frac{1}{2^{h+1}}) \bigg\rceil = \frac{1}{2} * \displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil \frac{h-1}{2^{h}} \bigg\rceil + \displaystyle\sum_{h=0}^{\lfloor logN \rfloor} \bigg\lceil \frac{1}{2^{h+1}} \bigg\rceil$
